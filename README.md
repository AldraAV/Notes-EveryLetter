# EveryLetter Sync Engine 🍒
> *"Quédate con las notas que te escribí porque cada letra las sentí muy en mi corazón :)"*

**Cada Letra** es un orquestador hiper-avanzado y plugin experimental de Obsidian que trasciende la simple transferencia de archivos. Basado en Algoritmos CRDT (Conflict-Free Replicated Data Types) y respaldado masivamente por la infraestructura de Supabase, Cada Letra provee Soberanía de Datos y Sincronización en Vivo para tus bóvedas.

## 🚀 Características (Nivel Dios)
- **Topología M2M (P2P Reactiva):** Cruce en tiempo real abstrayendo el documento físico a Matemática de Texto con `Yjs`.
- **Transmutación a BD:** Posibilidad de subir localmente el disco duro abstraído (Push Massivo) como base fundamental y ser absorbido (Pull Massivo) por dispositivos satélites para inicializar bóvedas de cero.
- **Centro de Mando Premium:** Dashboard estético con control de jerarquías para tu vault.
- **La Rutina del Cementerio (El No Olvido):** Si un invitado (satélite) intenta asesinar tu nota borrándola, el Oráculo anulará la instrucción de borrado definitivo `.trash()` interceptándola para meterla bajo sello en `.cada-letra-cementerio` con metadatos y timestamp.

## 🏛️ Instalación & Montaje a Mano
1. Clona este repositorio o descarga la release.
2. Extrae en tu carpeta de Obsidian `.obsidian/plugins/cada-letra-sync`.
3. Ejecuta `pnpm install`.
4. Ejecuta `pnpm run build`.
5. Activa y otorga los permisos correspondientes.
